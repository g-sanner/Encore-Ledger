## Installation Instructions

Follow these steps to download and run the project. <br/> 
<b>No technical background is required.</b>

### 1. Navigate to the Releases Section 
- Go to the **Releases** section of this GitHub repository.
- Located under the **About** section

<!-- Light mode image -->
<img src="assets/img/readme/1_LightMode.PNG#gh-light-mode-only" alt="Releases section - light mode">

<!-- Dark mode image -->
<img src="assets/img/readme/1_DarkMode.PNG#gh-dark-mode-only" alt="Releases section - dark mode">


### 2. Download the Latest Release
- Find the newest version at the top.
- Download the file named like:  
  `EncoreLedger-vX.X.X.zip`

<!-- Light mode image -->
<img src="assets/img/readme/2_LightMode.PNG#gh-light-mode-only" alt="Download section - light mode">

<!-- Dark mode image -->
<img src="assets/img/readme/2_DarkMode.PNG#gh-dark-mode-only" alt="Download section - dark mode">

### 3. Extract the ZIP File
- Locate the downloaded `.zip` file on your computer (usually in the **Downloads** folder).
- Right-click the file and select **Extract All…**
- Choose a folder where you want the project to be stored.
- Click **Extract**.

<img src="assets/img/readme/3_AllModes.PNG" alt="Releases section - light mode"> <br/>

<img src="assets/img/readme/4_AllModes.PNG" alt="Releases section - dark mode"> <br/>

<img src="assets/img/readme/5_AllModes.PNG" alt="Releases section - light mode">

### 4. Open the Project Folder
- After extraction, open the new folder that appears.
- You should see files such as:  
  - `EncoreLedger.exe` (or your main application file)  
  - Additional folders or resources included with the release

<img src="assets/img/readme/6_AllModes.PNG" alt="Releases section - light mode">

### 5. Run the Program
- Double-click the main application file to start the program.
- If Windows shows a security prompt, select **More Info** and then **Run Anyway**.

<img src="assets/img/readme/7_AllModes.PNG" alt="Releases section - light mode">

<img src="assets/img/readme/8_AllModes.PNG" alt="Releases section - dark mode">

### 6. Installation Complete
- The program should now open and run normally.
- If you experience issues, refer to the **Troubleshooting** section below.
